집밥의민족 웹사이트

집밥의민족은 레시피를 공유하고 요리 경험을 나누는 레시피 커뮤니티 웹사이트입니다. 집밥의민족은 누구나 집밥을 즐길 수 있도록 돕기 위해 시작되었습니다. 다양한 검색 기능으로 레시피를 쉽게 찾고 장보기 메모와 냉장고 파먹기로 요리를 준비합니다. 쩝쩝박사에서는 다양한 요리 주제로 소통하고 키친가이드와 쿠킹클래스에서는 요리 노하우를 나눕니다.

  • 작업 기간:   2025.02.07 - 04.20  &  2025.07.01 - 10.25
  • 작업 인원:   팀 프로젝트 (2인)
  • 작업 기여도:   70%
Java Spring Boot Spring Data JPA Spring Security REST API HTML CSS Javascript Oracle Linux Docker Jenkins AWS EC2
웹사이트 관리자사이트 깃허브 API 문서

프로젝트 개요

집밥의 민족은 다음과 같은 기능을 제공합니다.

  1. 다양한 재료 및 테마 기반 검색을 통해 원하는 레시피를 쉽게 찾아볼 수 있습니다.
  2. 장보기 메모, 냉장고 파먹기 등 부가 기능을 통해 요리 준비 과정을 더 편리하게 관리할 수 있습니다.
  3. 쩝쩝박사, 키친가이드, 커뮤니티 기능을 통해 사용자 간의 정보 공유와 상호작용이 가능합니다.

주요 담당 역할

  • 팀 협업 효율 개선
    • 개발 환경 세팅 및 기술 문서화(Figma, Notion, Wiki)를 주도하여 협업 효율성 향상
    • Git 브랜치 전략 수립 및 Swagger 기반 REST API 명세화로 팀 간 작업 충돌 최소화 및 구조 체계화
  • 데이터베이스 설계
    • Oracle 기반 ERD를 설계하고 이를 바탕으로 JPA 엔티티 구조를 구현하여 유지보수가 용이한 도메인 설계 달성
  • JWT 기반 인증 및 보안성 강화
    • Spring Security와 JWT를 적용하여 자체 로그인 및 OAuth2 소셜 로그인(Google, Naver 등) 기능 구현
    • Access/Refresh 토큰 분리 저장(로컬 스토리지 + 쿠키) 및 전역 예외처리를 통해 사용자 보안성과 사용자 경험 개선
    • 프론트엔드에서는 axios 인터셉터를 사용해 만료된 토큰을 감지하고 자동으로 재발급 요청을 수행
  • REST API 공통 응답 구조 및 예외 처리 설계
    • 공통 응답 포맷을 설계하고 커스텀 예외 클래스 및 글로벌 예외 처리를 구현
    • 클라이언트 일관성 있는 응답 구조 제공으로 프론트엔드 연동 효율 및 유지보수성 대폭 향상
  • 사용자 흐름에 기반한 UI/UX 기획 및 구현
    • Figma를 활용한 와이어프레임 설계
    • HTML/CSS/JavaScript을 활용하여 UI를 일부 구현하고 사용자 흐름과 시각적 집중도를 고려한 설계로 UX 향상
    • 공통 버튼/입력창 등 UI 컴포넌트를 재사용 가능한 형태로 직접 설계/제작하여 개발 효율성 향상
    • 회원가입 폼에서 AJAX 기반 실시간 유효성 검사를 구현하고, 입력창 하단에 사용자 친화적인 피드백 제공
  • 음식 기반 참여형 콘텐츠(쩝쩝박사) 제공
    • 투표, 매거진, 이벤트, 댓글 기능을 포함한 인터랙티브 콘텐츠 구성으로 사용자 참여 유도
    • Spring Data JPA의 Page<T>를 활용해 게시글과 댓글 목록에 페이징 처리를 적용하여 성능 최적화
  • 댓글 기능 제공
    • Spring Data JPA의 Page<T> 객체를 활용한 페이징 처리로 성능 최적화
    • 레시피, 쩝쩝박사, 키친가이드 등 다양한 게시판에 적용 가능한 공통 댓글 구조 설계

로그인 및 회원가입
  • 자체 로그인과 소셜 로그인
  • 이메일(SMTP) 기반 비밀번호 찾기
  • 사용자 닉네임 랜덤 추천
  • 회원가입 시 AJAX를 이용하여 입력창 하단에 실시간으로 폼값 검증 결과를 표시
레시피
  • 레시피 조회 및 작성 가능
  • 난이도, 조리시간, 맵기 정보 표시
  • 인분 수에 따른 재료 양 자동 조정
  • 장보기 메모에 간편하게 재료 담기 가능
  • 리뷰 및 댓글 기능
키친가이드
  • 손질법, 보관법, 요리팁 등을 모은 정보 게시판
  • 헤더 슬라이드 키워드 클릭 시 해당 게시글로 이동 가능
  • 관리자만 작성 가능하고 모든 사용자 댓글 작성 가능
쩝쩝박사
  • 음식 관련 투표, 매거진, 이벤트 제공
  • 댓글 기능 제공
쿠킹클래스
  • 사용자가 쿠킹클래스 개설을 신청하면 관리자가 승인 후 개설
  • 클래스 신청자는 간단한 신청서를 작성
  • 개설자는 신청자 관리 및 출석 체크 가능
  • 3회 이상 결석 시 마지막 신청일 기준 3개월간 신청 불가
후원
  • 포인트 충전 시스템
  • 마음에 드는 레시피 작성자에게 후원 가능
나의 냉장고와 냉장고 파먹기
  • 식재료를 육류 / 채소류 / 소스류 등으로 분류하여 냉장고에 등록 가능
  • 장보기 메모 작성 가능
  • 냉장고 속 재료 기반 요리 추천
  • Kakaomap을 활용한 근처 마트 위치 안내
